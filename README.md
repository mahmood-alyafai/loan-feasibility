# دراسة جدوى القروض - بنك التنمية العماني

**Development Bank of Oman - Loan Feasibility Study System**

---

## 🎯 Overview

An interactive web-based loan feasibility study application designed for **Development Bank of Oman** analysts to evaluate SME loan applications, generate financial projections, and produce standardized reports aligned with the bank's internal credit committee format.

---

## ✨ Features

| Section | Description |
|---------|-------------|
| **1. Borrower Data** | Company details, shareholders, investor CV, bank history |
| **2. Document Upload** | 10 required document upload fields (ID, CR, Malaa, financial statements, bank statements, quotations, site visit report, etc.) |
| **3. Project & Resources** | Project description, HR table, site details, SWOT analysis |
| **4. Costs & Financial Analysis** | Investment cost breakdown, financing structure, loan terms, income statement, balance sheet, financial ratios |
| **5. Scoring & Recommendation** | Auto-calculated credit score (0-100), risk classification, final recommendation (Approve / Conditional / Reject) |
| **6. Report Preview** | Printable report matching the bank's official format with DBO branding |
